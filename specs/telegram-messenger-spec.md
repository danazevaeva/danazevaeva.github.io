# Спецификация: Telegram-мессенджер для брокера ATON

## Обзор

Telegram-подобный мессенджер внутри брокерского приложения ATON. Неавторизованная зона для потенциальных клиентов, которые ещё не открыли счёт. Цель — прогрев экспертным контентом и конверсия в открытие счёта.

- **Платформа:** iOS (iPhone)
- **Viewport:** 393×852px
- **Тема:** Светлая (white theme ATON)
- **Стиль:** Премиальный, минималистичный, дизайн-система ATON

## Экран 1: Лента канала

### Структура (сверху вниз)

#### 1.1 Status bar — 54px
- Системный статус-бар iOS

#### 1.2 Хэдер канала — 56px
- **Слева:** Аватарка канала (36×36px, круглая, градиент #7000FF → #202020)
- **По центру-лево:** Название "ATON Investments" (Gotham Bold 16px, #202020) + подписчики (12px, #8D9398)
- **Тап по аватарке/названию** → переход в Профиль канала (Экран 2)
- **Разделитель внизу:** 1px solid #E3E9EF

#### 1.3 Плашка закреплённых сообщений — 44px
- **Фон:** #F8FBFD
- **Левый бордер:** 3px solid #7000FF
- **Строка 1:** "Закреплённое сообщение 1 из N" (11px, #7000FF, font-weight:600)
- **Строка 2:** Текст закрепа (13px, #202020, однострочный, text-overflow:ellipsis)
- **Справа:** Стрелка вниз для переключения (▼, #8D9398)
- **Поведение:** Тап — переключает на следующий закреп (циклически). Свайп вверх — переход к закреплённому сообщению в ленте

#### 1.4 Лента постов — скроллируемая область
Занимает пространство между плашкой закрепов и CTA-кнопками.

**Типы постов:**

##### Пост: Текст + Изображение
- **Автор:** Аватарка (28×28px) + Имя канала (Gotham Medium 14px, bold) + Время (12px, #8D9398)
- **Изображение:** На всю ширину (padding 16px с боков), border-radius:12px, высота пропорциональная
- **Текст:** Под изображением, 14px/22px line-height, #202020. Заголовок bold, описание #8D9398
- **Метрики:** Просмотры + комментарии (12px, #8D9398), иконки + числа
- **Отступ между постами:** 20px

##### Пост: Только текст
- Аналогично, без блока изображения

##### Пост: Карусель изображений
- Несколько изображений — свайп горизонтально (как в Telegram, полная ширина)
- Индикатор точек внизу изображения

##### Пост: Опрос
- **Вопрос:** 14px, #202020
- **Варианты ответов:** Прямоугольники bg:#EFF3F7, border-radius:10px, padding:10px 14px
- **Прогресс-бар:** Фон #E8E2FF (c13), ширина пропорциональна проценту
- **Проценты:** Текст поверх прогресс-бара (13px)
- **Счётчик голосов:** Под вариантами (12px, #8D9398)
- **Gap между вариантами:** 6px

#### 1.5 Плавающая плашка CTA — 80px + 34px safe area
- **Фон:** rgba(255,255,255,0.85) + backdrop-filter:blur(20px)
- **Padding:** 12px 16px сверху/бока, 46px снизу (включая safe area 34px)
- **Две кнопки в ряд (50/50), gap:8px:**
  - **"Открыть счёт"** — `.btn-main`: bg:#202020, color:#fff, h:48px, r:16px, Gotham Medium 16px
  - **"Связаться"** — `.btn-secondary`: bg:#EFF3F7, color:#202020, h:48px, r:16px, Gotham Medium 16px

## Поведение и анимации

### Переходы между экранами
- Лента → Профиль: push справа (стандартный iOS navigation transition)
- Профиль → Лента: pop слева

### Скролл ленты
- Лента скроллится свободно
- Плашка закрепов фиксирована вверху
- CTA-кнопки фиксированы внизу
- Контент уходит под blur CTA-плашки

### Закреплённые сообщения
- Тап по плашке — переключение на следующий закреп (1→2→3→1)
- Анимация: fade-crossfade текста (200ms)

### Опросы
- Тап по варианту — анимация прогресс-бара (300ms ease-out)
- После голосования видны проценты всех вариантов
